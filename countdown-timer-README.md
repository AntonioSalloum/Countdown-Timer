# Countdown Timer Using TimerTask:
A simple Java program built to practice scheduling tasks using the Timer and TimerTask classes.
Counts down from 10 and prints a New Year message when it reaches 0.
# Features:
- Countdown from 10 to 0
- Prints a message once the countdown ends
- Automatically stops the timer after completion
# How to Run:
- Open terminal
- move to project repository
- write: javac Main.java
- write: java Main
- watch the countdown run automatically
# Requirements:
- Java JDK 8 and above
# Concepts Practiced:
- Java Timer and TimerTask classes
- Anonymous Inner Classes
- Scheduled Task Execution
- Basic Fundamentals Logic Like: Conditionals, Counters
